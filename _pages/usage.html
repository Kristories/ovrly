Usage

Using LESS

less 
.foo{
    .ovrly-pixel();
}

.bar{
    .ovrly-pixel(#ED5821);
}
html 
<div class="foo">
    OVRLY
</div>

<div class="bar">
    OVRLY with color
</div>

Using CSS

html 
<div class="ovrly-pixel">
    OVRLY
</div>

Color

Default color is #333333, but you can change .ovrly color from here.
