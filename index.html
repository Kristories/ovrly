Installation

Bower

Make sure you have bower installed or install it via npm install -g bower.

console 
bower install ovrly --save
less 
@import "bower_components/ovrly/less/ovrly";

Old School

Download ovrly.css or ovrly.min.css.

html 
<link rel="stylesheet" href="path/to/ovrly.css">

Usage

Using LESS

less 
.foo{
    .ovrly-hexagon();
}

.bar{
    .ovrly-hexagon(#ED5821);
}
html 
<div class="foo">
    OVRLY
</div>

<div class="bar">
    OVRLY with color
</div>

Using CSS

html 
<div class="ovrly-hexagon">
    OVRLY
</div>

Color

Default color is #333333, but you can change .ovrly color from here.
